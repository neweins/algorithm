## theory

1. 그리디 알고리즘
- 모든 관계가 배수 관계가 성립하면 그리디 알고리즘 사용 가능하다.
  ex> 동전 1, 5, 50원 중 선택해서 최소 또는 최대 동전 수를 내야할 때 1, 5, 50은 서로 배수 관계이다.
